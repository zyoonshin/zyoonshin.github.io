Variables (변수)

  • 프로그램을 실행하는 동안 값이 여러 번 달라질 수 있는 데이터

    • ↔ constant(상수) : 값을 한번 지정하면 바뀌지 않는 데이터


변수 선언의 규칙

  • 영어 문자와 언더 스코어(_), 숫자 사용
    • now, _now, now25, now_25 ← 사용 가능
    • 25now, now 25, *now ← 사용 불가능

  • 영어 대소 문자를 구별하며, 예약어는 사용 불가능
    • number, Number, NumBer 모두 다른 변수로 인식
    • var처럼 미리 정해 놓은 예약어는 변수 이름으로 사용 불가능

  • 여러 단어를 연결한 변수 이름은 중간에 대문자
    • totalArea, TotalArea, Total_Area ← 낙타 표기법(camel case)

  • 의미 있게 작성
    • 변수 이름만 보고도 대충 어떤 값인지 추측 가능하도록

변수 선언하기

						var 변수명

Data type (자료형)

  • 컴퓨터가 처리할 수 있는 자료의 형태
    • 컴퓨터에게 일을 시킬 때 ‘10’은 숫자이고 ‘안녕하세요’는 문자라는 것을 따로 알려줘야 처리 가능

표 만들기

Number (숫자형)

  • 정수와 실수로 나누어 구분
    • 정수 : 소수점이 없는 숫자
    • 실수 : 소수점이 있는 숫자

      • → 자바 스크립트에서는 정수와 실수를 함께 묶어 숫자형이라 정의
      → 정밀하게 숫자를 계산하는 프로그램을 만들때는 주의

ex) 0.1 + 0.2 = 0.300000000000000000000000004
자바 스크립트에서는 0.1이나 0.2를 2진수로 변환해서 계산하는데 이때 자릿수가 많은 소수로 변환되고, 그 상태에서 0.1과 0.2를 더하기 때문에 0.3000000000004 출력


String (문자열)

  • 작은 따옴표(’’)나 큰 따옴표(””)로 묶은 데이터 의미
  • 숫자도 작은 따옴표나 큰 따옴표로 묶으면 문자열로 인식함
    • 큰 따옴표로 묶은 문자열 안에 또 다른 문자열을 넣으려면?

      • → 큰 따옴표가 중복되지 않도록 속성 값은 작은 따옴표로 묶어서 표현


Boolan (논리형)

  • 참(true)이나 거짓(false)의 값을 표현하는 자료형
  • 어떤 조건을 확인해서 그 조건이 맞으면 true, 맞지 않으면 false

Undefined / Null

  • undefined
    • 자료형이 정의되지 않았을 때의 데이터 상태
    • 선언만 하고 값 할당 X
  • null
    • 데이터의 값이 유효하지 않은 상태

Array (배열)

  • 하나의 변수에 값을 여러 개 저장
  • 여러 개의 데이터 값을 하나의 배열 이름으로 묶어서 선언
    • 데이터 값을 쉼표로 구분해서 대괄호([])로 묶으면 배열 선언
    • 빈 배열도 선언 가능
						배열명["값1", "값2", ...]
배열명[]

  • 배열 사용 X
var spring = "봄";
var spring = "여름";
var spring = "가을";
var spring = "겨울";
  • 배열 사용 O
var season = ["봄", "여름", "가을", "겨울"];
