Java Concept

Published 2 days ago5 min read 4 comments

Java란?

  • 웹 애플리케이션 코딩에 널리 사용되는 프로그래밍 언어
  • 그 자체로 플랫폼으로 사용할 수 있는 다중 플랫폼, 객체 지향 및 네트워크 중심 언어
  • 모든 것을 코딩하기 위한 빠르고 안전하며 안정적인 프로그래밍 언어
  • 지역화된 분산 소프트웨어 구축
  • WORA(Write Once and Run Anywhere)라고도 함

Java 프로그래밍 용도

1. 게임 개발
  • 기계 학습이나 가상 현실과 같은 첨단 기술이 통합된 최신 게임 작성
2. 클라우드 컴퓨팅
  • 분산 클라우드 기반 애플리케이션에 적합
  • 클라우드 공급자는 다양한 기본 플랫폼에서 프로그램을 실행하기 위해 Java 언어 선택
3. 빅 데이터
  • 복잡한 데이터 집합과 방대한 양의 실시간 데이터를 처리할 수 있는 데이터 처리 엔진에 사용
4. 인공 지능
  • 기계 학습 라이브러리의 발전소 역할
  • 자연어 처리 및 딥 러닝과 같은 인공 지능 애플리케이션 개발에 적합한 안정성과 속도를 갖춤
5. 사물 인터넷
  • 인터넷에 독립적으로 연결할 수 있는 엣지 디바이스의 센서 및 하드웨어 프로그래밍에 사용

Java가 인기있는 이유

고품질 학습 리소스
  • 오랫동안 사용되어 왔기 때문에 새로운 프로그래머가 사용할 수 있는 학습 리소스 多
기본 제공 함수 및 라이브러리
  • 개발자가 처음부터 모든 새로운 기능을 작성할 필요 X
  • 다양한 애플리케이션을 개발하기 위해 기본 제공 기능과 라이브러리로 구성된 풍부한 에코시스템 제공
적극적인 커뮤니티 지원
  • 코딩 문제 발생 시 개발자를 지원할 수 있는 많은 활성 사용자와 커뮤니티 O
  • Java 플랫폼 소프트웨어도 정기적으로 유지 관리되고 업데이트됨
고품질 개발 도구
  • 자동화 편집, 디버깅, 테스트, 배포 및 변경 관리를 지원하는 다양한 도구 제공
  • Java 프로그래밍을 시간과 비용 효율적으로 만듦
플랫폼 독립적
  • Windows, Linux, iOS 또는 Android와 같은 기본 플랫폼에서 실행 가능
  • 여러 디바이스에서 애플리케이션을 실행하려는 오늘날의 환경에서 특히 유용
보안
  • 신뢰할 수 없는 코드가 호스트 시스템을 바이러스에 감염시킬 수 X
  • 하드 드라이브에서 파일을 읽거나 쓸 수 X
  • Java의 보안 수준 및 제한 사항도 고도로 구성 가능
